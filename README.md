# 🚀 HybridHire AI

HybridHire AI is an AI-powered recruitment and career assistance platform designed to simplify the hiring process for recruiters and provide career guidance to students.

The platform helps recruiters analyze resumes, evaluate candidates, and make better hiring decisions using intelligent screening insights. It also provides students with resume analysis and career improvement guidance.

---

## ✨ Features

### 👨‍💼 Recruiter Module
- Recruiter dashboard with hiring insights
- Upload and analyze bulk resumes
- Hybrid AI Score Evaluation for candidate assessment
- Intelligent resume analysis and candidate ranking
- Candidate screening management
- Export recruitment reports

### 👨‍🎓 Student Module
- Resume upload and analysis
- Resume improvement suggestions
- Career guidance based on skills and profile
- Skill development recommendations
- Certificate and application tracking
- - AI-driven job-role matching
